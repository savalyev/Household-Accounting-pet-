# Домашняя бухгалтерия (Pet-проект)

Этот репозиторий содержит личный pet-проект для отслеживания расходов и бюджета домохозяйства. Простой инструмент для управления семейными финансами, помогающий контролировать траты по категориям: продукты, коммуналка, развлечения и многое другое.

## Возможности
Категоризация расходов: Отслеживание трат по заранее заданным категориям домохозяйства (еда, счета, транспорт, развлечения).
Планирование бюджета: Установка месячных лимитов с визуализацией превышений.
Отчетность: Генерация сводок и графиков трендов расходов.
Экспорт данных: Выгрузка записей в CSV для дальнейшего анализа.


# Быстрый старт
## Клонируй репозиторий
git clone https://github.com/savalyev/Household-Accounting-pet-.git
cd Household-Accounting-pet-

## Установи зависимости
npm install

## Создать БД и пользователя:
createdb -U postgres your_db_name, createuser -U postgres your_user.

## Настроить:
.env из .env.example.

## Импортировать: 
psql -U your_user -d your_db_name -f init.sql.

## Запуск
npm run dev
cd backend && node server.js
